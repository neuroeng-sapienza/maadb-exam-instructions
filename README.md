# Exam instructions for the MAADB exam

View the pages locally:

```sh
uv run mkdocs serve
uv run mkdocs serve -o
uv run mkdocs serve -o -a localhost:8001
```

Publish the pages:

uv run mkdocs gh-deploy

The published pages are available at:
